Д-30



по умолчанию при постройке имеет по 30 снарядов разных типов:
а) осколочно-фугасные (30Ф56), предназначены для уничтожения на больших площадях пехоты и лёгкой бронетехники противника, хотя неплохо повреждают и танки (стоят по умолчанию);
б) 122-мм (дым), для установки дымовой завесы;
в) 122-мм СПБЭ (самоприцеливающийся боевой элемент), против танков;
г) 122-мм (лазер), предназначены для стрельбы по целям, высвечиваемым лазерными целеуказателями (из снаряжения игрока);
д) Д-462 (дым), для установки дымовой завесы;
е) С-463 (осветительные), для освещения заданного участка в ночное время суток.
